B-29 Superfortress (Enola Gay)

Version 1

Extremely detailed B-29 depicting the Enola Gay airframe as it appeared on August 6th, 1945. Features full cockpit, detailed bomb bays with atomic bomb mounting hardware and gunners section. Includes separate flight control surfaces, landing gear, both open and closed landing gear doors, bomb bay doors, propellers, and gun turrets for animating.

Model and textures by Marv Mays.
