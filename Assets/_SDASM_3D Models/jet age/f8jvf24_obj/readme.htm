F-8J Crusader

Version 1

Extremely detailed Vietnam era US Navy fighter. Includes separate double-hung slats, positionable wing, tails, rudder, canopy, seat and landing gear for animating. Model features very high resolution texture maps for close-up rendering.

Note: The tailhook must be rotated into the fully raised position for correct in-flight animation.

Model and textures by Marv Mays.
