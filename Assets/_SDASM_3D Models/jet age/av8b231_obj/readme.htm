AV-8B Harrier II (VMA-231 Desert Storm)

Version 1

This is a detailed AV-8B Harrier jet from VMA-231 in the USMC evaluation paint scheme the squadron wore during Operation Desert Storm in 1991. The model features separate canopy, ejection seat, gun pod, hot & cold exhaust nozzles, flight control surfaces, and landing gear for animating.

Model and textures by Marv Mays.
